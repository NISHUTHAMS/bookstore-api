# bookstore-api
A RESTful Bookstore API built with Node.js, Express, and MongoDB, providing full CRUD functionality for managing books.

## Features
- Add a new book
- Get all books
- Get a book by ID
- Update a book
- Delete a book

## Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)

## Run Locally
```bash
npm install
npm start
